原生JS向指定dom元素尾部插入dom节点

· 2020-08-26 11:05:58 · # JS

接上文：原生JS将字符串转换为dom节点.

如何使用原生JS向指定dom元素尾部插入dom节点呢？这里介绍两种方式：

方案一：appendChild

使用方法：

// eg:
document.body.appendChild(element);

需要注意的是，appendChild 方法只能传入 1个 dom元素。

方案二：innerHTML

使用方法：

let bodyHtml = document.body.innerHTML;
bodyHtml += element;
document.body.innerHTML = bodyHtml;
